## Principes de Développement
- **Typage fort (Type-Safety)** : TypeScript par défaut pour des bases de code robustes et maintenables.
- **Performance** : Philosophie du "Moins, c'est plus" (*Less is more*). Je privilégie les fonctionnalités web natives et le CSS plutôt que le JavaScript lourd.
- **Éthique & Confidentialité** : Conformité RGPD native, accessibilité (A11y) et respect des standards du web.
